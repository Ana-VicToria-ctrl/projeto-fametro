Sobre o projeto

Um portal pensado para valorizar Tefé com mais presença visual e melhor organização

Este trabalho junta turismo, identidade cultural e sistema web em uma proposta mais completa. A ideia foi deixar o projeto bonito para apresentar e funcional no uso real.

  • Visual mais caprichado e leitura mais confortável
  • Painel para visitantes e acesso administrativo separado
  • Gerenciamento de atrações e sugestões em um só lugar
Projeto acadêmico

Ana Vitória

Projeto desenvolvido com foco em HTML, CSS, experiência do usuário e integração com backend em Flask.

Objetivo

Apresentar os principais pontos turísticos de Tefé-AM de forma bonita, clara e fácil de navegar.

Tecnologias

Flask, SQLite, HTML, CSS e templates Jinja2 trabalhando juntos em uma solução leve e funcional.

Resultado

Um site que deixou de ser só vitrine estática e passou a ter login, painel e gerenciamento de conteúdo.
